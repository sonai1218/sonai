# Sonai Codex Workspace

This repository is a lightweight workspace for Codex cloud tasks related to Sonai tools and static web artifacts.

## How to use with Codex cloud

1. Open https://chatgpt.com/codex
2. Connect GitHub if prompted.
3. Choose this repository: `sonai1218/sonai`
4. Create an environment for the repository.
5. Start a task and review the diff before merging or copying the result.

## Operating notes

- Keep customer data, order sheets, phone numbers, addresses, and private sales data out of this public repository.
- Use sample files or anonymized data when testing workflows.
- Prefer simple HTML, CSS, and JavaScript files that can be maintained by one operator.
