# Sonai Codex Context

이 파일은 Codex Cloud가 작업 전에 읽는 기본 맥락 파일입니다.
공개 저장소에 들어가는 파일이므로 고객정보, 주문정보, 내부 매출자료, 계정정보는 적지 않습니다.

## 기본 역할

소나이는 남성복 편집샵입니다. 단순한 의류 판매보다, 장인 기반 브랜드와 덜 알려졌지만 설득 가능한 브랜드를 발굴하고 그 맥락을 고객에게 설명하는 큐레이션형 편집샵입니다.

Codex는 소나이 운영자가 실제로 쓸 수 있는 작고 명확한 결과물을 만드는 데 집중해야 합니다.

## 작업 기준

- 1인 운영자가 관리할 수 있어야 합니다.
- 복잡한 프레임워크보다 단순한 HTML, CSS, JavaScript 구조를 우선합니다.
- 외부 라이브러리는 꼭 필요할 때만 사용합니다.
- 텍스트, 색상, 이미지 경로, 옵션명은 나중에 쉽게 바꿀 수 있어야 합니다.
- 고객정보와 민감한 자료는 GitHub에 올리지 않습니다.
- 불확실한 값은 추측하지 말고 `확인 필요`라고 표시합니다.

## 문체 기준

- 기본 문체는 `합니다 / 했습니다`체입니다.
- 과장된 홍보 문구를 피합니다.
- 장인정신을 감상적으로 미화하지 않습니다.
- 고객에게 판단 근거를 나누는 방식으로 설명합니다.
- 긴 대시, 과한 특수문자, 이모티콘은 사용하지 않습니다.

## 코드 결과물 기준

- 가능하면 하나의 HTML 파일로 작동하게 만듭니다.
- 비개발자도 수정할 수 있도록 필요한 곳에 짧은 주석을 답니다.
- 이미지 경로, 컬러값, 주요 문구는 찾기 쉬운 위치에 둡니다.
- 작업 후에는 무엇을 만들었는지, 어떤 파일을 수정했는지, 어디를 바꾸면 되는지, 어떻게 실행하는지 쉽게 설명합니다.

## 새 작업 시작 문장

Codex Cloud에서 새 작업을 시작할 때 아래 문장을 먼저 붙여넣습니다.

```text
AGENTS.md와 context.md를 먼저 읽고, 그 기준에 맞춰 작업해줘.
나는 비개발자라서 결과물은 쉽게 수정할 수 있는 구조로 만들어줘.
가능하면 HTML, CSS, JavaScript 기반의 단순한 파일로 만들어주고, 고객정보나 민감한 자료는 GitHub에 올리지 말아줘.
작업이 끝나면 무엇을 만들었는지, 어떤 파일을 바꾸면 되는지, 어떻게 열면 되는지 쉽게 설명해줘.
```
